$ weather_click

  weather_click application 
  collect for you forcast up to the next 4 days
  for any city you wish for.


Usage:

  $ pip install --editable .
  
  $ examples:
    
    weather_click --city "tel aviv"
    
    weather_click --city rome --forecast 4 -f
